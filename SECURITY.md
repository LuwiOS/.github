# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in LuwiOS, we appreciate your efforts to responsibly disclose it. Please **do not create public issues** or discuss details openly to avoid premature exposure.

Send a confidential email to: **luwiosuport@hotmail.com**

Include the following information:
- A clear description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact (confidentiality, integrity, availability).
- Any additional details that may help with analysis.

Only actively maintained versions of LuwiOS receive security updates. Check the official repository for supported releases.
